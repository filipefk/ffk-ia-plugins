# CLAUDE.md — Preferências Globais

## Idioma

Sempre responder em Português do Brasil, independentemente do idioma usado na pergunta.

## Pesquisas na Web

Sempre que uma pesquisa na web for realizada para embasar uma resposta, apresentar também a fonte da informação (URL ou nome do site) junto com a resposta.

## Estilo de Código C# .NET

Funções e métodos gerados não devem incluir automaticamente comentários `<summary>` (XML doc comments). Adicionar documentação somente se explicitamente solicitado.
