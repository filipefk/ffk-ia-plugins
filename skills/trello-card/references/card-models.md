# Modelos de cards (Markdown)

O Trello renderiza Markdown no campo `desc`. Use os modelos abaixo como base.

## Story

```markdown
**Como** [ator inferido], **quero** [ação principal] **para** [benefício].

## Critérios de Aceite
- [critério 1 extraído da descrição]
- [critério 2, se houver]
```

## Bug

```markdown
## Comportamento Atual
[o que está acontecendo de errado, conforme descrito]

## Comportamento Esperado
[o que deveria acontecer]

## Passos para Reproduzir
1. [passo 1, se mencionado]
```

## Task

```markdown
## Objetivo
[o que deve ser feito]

## Detalhes
[informações relevantes da descrição]
```
